# E-commerce

In my project, I handled the cart using a simple JavaScript array.
Whenever a user clicks on ‘Add to Cart’, I check if that product is already in the cart. If it is, I just increase its quantity, otherwise I add it as a new item.

To make sure the cart data doesn’t get lost after refreshing the page, I used localStorage. I save the cart data in localStorage, and whenever the page loads, I get that data back and show it again.

This way, the user’s cart stays saved even after refresh.

Tech Stack: HTML, CSS, JavaScript
